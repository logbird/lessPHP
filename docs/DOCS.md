框架使用说明
=======

内置模版引擎语法
-------

`<!--{ 语法区域 }-->`

1. 获取常量

```
	&lt;!--{C:常量名}--&gt;
```

2. 获取变量

```
	&lt;!--{$变量名}--&gt;
```

3. 循环

```
	&lt;!--{foreach as $k => $v}--&gt;
		循环体
	&lt;!--{/foreach}--&gt;
	
	&lt;!--{for $i=0; $i < 10;$++}--&gt;
		循环体
	&lt;!--{/for}--&gt;
```

4. 调用函数

```
	&lt;!--{P:函数名}--&gt;
```

5. if语句

```
	&lt;!--{if 表达式}--&gt;
	&lt;!--{/if}--&gt;
```
