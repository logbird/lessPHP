框架使用说明
=======

内置模版引擎语法
-------

`<!--{ 语法区域 }-->`

1. 获取常量
	&gt;!--{C:常量名}--&lt;
2. 获取变量
	&gt;!--{$变量名}--&lt;
3. 循环
	&gt;!--{foreach as $k => $v}--&lt;
		循环体
	&gt;!--{/foreach}--&lt;
	
	&gt;!--{for $i=0; $i < 10;$++}--&lt;
		循环体
	&gt;!--{/for}--&lt;
4. 调用函数
	&gt;!--{P:函数名}--&lt;
5. if语句
	&gt;!--{if 表达式}--&lt;
	&gt;!--{/if}--&lt;
