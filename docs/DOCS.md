内置模版引擎语法
-------

`<!--{ 语法区域 }-->`

1. 获取常量

```
	<!--{C:常量名}-->
```

2. 获取变量

```
	<!--{$变量名}-->
```

3. 循环

```
	<!--{foreach as $k => $v}-->
		循环体
	<!--{/foreach}-->
	
	<!--{for $i=0; $i < 10;$++}-->
		循环体
	<!--{/for}-->
```

4. 调用函数

```
	<!--{P:函数名}-->
```

5. if语句

```
	<!--{if 表达式}-->
	<!--{/if}-->
```
